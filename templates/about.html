This project focuses on developing a machine learning model capable of accurately predicting movie genres based on their plot summaries. The scope includes:


Capstone Project Scope


Data Collection and Preprocessing

Gathering a diverse dataset of movie plot summaries and their corresponding genres, followed by data cleaning, preprocessing, and feature engineering.


Model Development

Building, training, and fine-tuning various machine learning models (e.g., Naive Bayes, Support Vector Machines, Random Forest, Deep Learning) to classify movie genres.


Model Evaluation

Assessing the performance of the developed models using appropriate metrics (e.g., accuracy, precision, recall, F1-score) and selecting the best-performing model.


Model Deployment

Creating a user-friendly interface or API for integrating the model into recommendation systems or other applications.


Impact Analysis

Evaluating the impact of the genre prediction model on recommendation system performance through A/B testing or similar methods.
